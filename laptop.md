# Перелік компонентів ноутбука Acer Nitro 5
### Модель: Acer nitro 5 AN515-57-55MK 

Дана таблиця містить опис внутрішніх вузлів ноутбука на основі технічної схеми.

| № | Назва (укр) | Назва (en) | Опис / Роль у системі |
|:---:|:---|:---|:---|
| 1 | Материнська плата | Motherboard | Основна системна плата, на якій розміщені CPU та GPU |
| 2 | Процесор та Відеокарта | CPU & GPU (under heatsink) | Intel Core i5-11400H та NVIDIA GTX 1650 |
| 3 | Система охолодження | Heatsink thermal | Мідні теплотрубки для відведення тепла |
| 4 | Вентилятор кулера | Fan | Активне охолодження Dual Fan |
| 5 | Акумуляторна батарея | Battery | Літій-іонний акумулятор для автономної роботи |
| 6 | Оперативна пам'ять | RAM | Модулі пам'яті DDR4 для Acer Nitro 5 |
| 7 | Твердотільний накопичувач M.2 | SSD M.2 Sata/NVMe | Основний накопичувач під ОС та файли |
| 8 | Модуль Wi-Fi | Wifi card | Бездротовий адаптер мережі |
| 9 | Плата вводу-виводу | IO board | Додаткова плата з портами USB та аудіо-роз'ємом |
| 10 | З'єднувальний шлейф плат | IO/MB Connecting cable | Магістральний кабель між основною та дочірньою платою |
| 11 | Динамік (правий) | Right speaker | Компонент стереосистеми |
| 12 | Динамік (лівий) | Left speaker | Компонент стереосистеми |
| 13 | Петля екрану (права) | Right hinge | Кріплення та механізм відкриття дисплея |
| 14 | Петля екрану (ліва) | Left hinge | Кріплення та механізм відкриття дисплея |
| 15 | Тачпад | Touchpad | Сенсорна панель керування курсором |
| 16 | Шлейф підсвітки | Backlight LVDS Cable | Шлейф для живлення підсвітки матриці |
| 17 | Шлейф клавіатури | Keyboard Connecting cable | Підключення клавішного блоку до плати |
| 18 | Антени Wi-Fi | Wifi antennas | Кабелі для прийому сигналу бездротового зв'язку |
| 19 | Шлейф сенсору | Touch screen LVDS cable | Опціональний шлейф для моделей з тач-скріном |
| 20 | Топкейс (верхня частина) | Topcase | Лицьова панель корпусу навколо клавіатури |

---
*Примітка: Перелік складено відповідно до візуальної схеми компонування та технічних характеристик моделі Acer Nitro 5.*
