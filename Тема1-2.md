# ТЕМА 1–2  
## Апаратне забезпечення персонального комп’ютера. Електробезпека

---

## 1. Правила електробезпеки при роботі з внутрішніми компонентами ПК

Робота з апаратними компонентами персонального комп’ютера пов’язана з ризиком ураження електричним струмом та пошкодженням електронних елементів. Для безпечної роботи необхідно суворо дотримуватись правил електробезпеки.

### Основні вимоги електробезпеки:
- повне **відключення ПК від електромережі** перед початком робіт;
- зняття залишкової напруги шляхом натискання кнопки живлення;
- робота **сухими руками** в сухому приміщенні;
- заборона дотику до контактів, доріжок та мікросхем;
- використання **антистатичного захисту** (браслет, заземлення);
- заборона роботи з пошкодженими кабелями та роз’ємами.

> ⚠️ **Недотримання правил електробезпеки може призвести до виходу з ладу компонентів або травмування людини.**

## 2. Форм-фактори корпусів настільних комп’ютерів

Форм-фактор корпусу визначає його габаритні розміри, внутрішнє компонування, систему охолодження та сумісність із материнськими платами.

### Основні типи корпусів:

| Форм-фактор | Характеристика | Призначення |
|------------|----------------|-------------|
| **Full Tower** | Великі розміри, максимальна вентиляція | Сервери, ігрові ПК |
| **Mid Tower** | Стандартний розмір | Домашні та навчальні ПК |
| **Mini Tower** | Компактний | Офісні системи |
| **Desktop (Slim)** | Горизонтальне розміщення | Каси, офіси |

### Переваги правильного вибору корпусу:
- ефективне охолодження компонентів;
- зручність монтажу та обслуговування;
- можливість подальшої модернізації.

## 3. Блок живлення: характеристики та конектори

Блок живлення (Power Supply Unit, PSU) призначений для перетворення змінної напруги електромережі у постійну напругу, необхідну для живлення компонентів ПК.

### Основні технічні характеристики PSU:
- **Номінальна потужність**: 400–1000 Вт;
- **Коефіцієнт корисної дії (ККД)** — сертифікація 80 PLUS;
- **Вхідна напруга**: 220–240 В;
- **Системи захисту**: від короткого замикання, перенапруги, перегріву.

### Основні конектори блоку живлення:

| Конектор | Призначення |
|--------|-------------|
| 24-pin ATX | Живлення материнської плати |
| 4/8-pin CPU | Живлення процесора |
| PCIe 6/8-pin | Живлення відеокарти |
| SATA | HDD, SSD, оптичні приводи |
| Molex | Додаткові пристрої |

## 4. Основні компоненти материнської плати (7 шт.)

Материнська плата є центральним елементом комп’ютерної системи, що забезпечує взаємодію всіх компонентів.

### Основні компоненти материнської плати:
1. **Процесорний роз’єм (Socket)** — для встановлення центрального процесора  
2. **Чипсет** — керування обміном даних між компонентами  
3. **Слоти оперативної пам’яті (DIMM)**  
4. **Слоти розширення (PCI Express)**  
5. **Роз’єми живлення**  
6. **Мікросхема BIOS / UEFI**  
7. **Панель введення-виведення (USB, LAN, Audio)**

---

## Висновок

Знання будови персонального комп’ютера та дотримання правил електробезпеки є обов’язковими умовами безпечної, надійної та ефективної роботи з комп’ютерною технікою.
